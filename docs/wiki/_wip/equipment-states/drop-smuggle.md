# Drop Smuggle

A smuggle in the drop [equip position](uid:S8I).

Note that [drop _equips_](../drop-equip/) are sometimes referred to as drop smuggles, despite not being true smuggles.

## Methods

[Search for drop smuggle methods](search:Drop Smuggle)

## Converting to a zuggle

Equip and warm-drop any other equipment of the same type. The drop smuggle will become a drop zuggle.

## Undoing

To undo a drop smuggle, equip it and fail drop it.

Unlike with other equip positions, dropping, throwing, or shock-dropping active equipment doesn't affect drop smuggles.
