# Dynamic Smuggle

A smuggle in the dynamic [equip position](uid:S8I).

## Methods

[Search for dynamic smuggle methods](search:Dynamic Smuggle)

## Converting to a zuggle

Equip and then warm-drop (or fail drop) any other equipment of the same type. The dynamic smuggle will become a dynamic zuggle.

## Undoing

Dynamic smuggles are released when some active equipment of the same type is thrown or shock-dropped.
