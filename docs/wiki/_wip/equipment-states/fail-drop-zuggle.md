# Fail Drop Zuggle

This is just a zuggled [Fail Drop Equip](uid:F4E).

## Effects

Fail drop zuggles are exactly like fail drop equips, except:

- They persist through reloads just like any zuggle.
- The persistent prompt effect is not lost when Link changes how he holds that equipment type.
- Warping doesn't undo this state either.
- Equipping a fail drop zuggle will equip in the [static position](uid:S8I). To revert it to the normal dynamic position, fail drop it normally.
    - This method of static equip still keeps the pickup prompt, so you can pick up a second copy of the item, unloading its actor.
    - It also keeps the sparkle effect of nearby dropped items.
