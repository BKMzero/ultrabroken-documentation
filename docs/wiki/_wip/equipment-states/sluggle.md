# Sluggle

Sluggling (sometimes written as _Slugging_) is a set of glitches involving a piece of equipment being eaten by a Like Like, and also equipped.

When Like Likes eat equipment, they don't fully delete it (for the most part), they just cull it. Killing the Like Like, or making it spit back out, will then uncull it again. Sluggle isn't so much a separate equipment state as it is a convenient cull/uncull source, when other culling options aren't sufficient.
