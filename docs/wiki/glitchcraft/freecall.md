# Freecall

## Summary
Breaks the lock on a recall lock, allowing the object to be in freecalled state.

_suusi, ROBUXY2ND - 9 September 2023_

## Instructions
1. Recall lock an object
2. Attach an object to it
3. Recall said object
4. Break off the object with Ultrahand
