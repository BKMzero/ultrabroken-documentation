# Mineru Parented Zuggle

Creates a zuggle by invizuggling it, then using an existing Mineru FE connection to uncull that zuggle in place. This provides a convenient way to create an unculled zuggle on 1.2.0+, without a pause buffer.

## Instructions

1. Equip the item to zuggle, which must be unfused.
1. Drop it from Link with any method.
1. [Fuse entangle it to Mineru](uid:IG2).
1. Pick up the item. If you want to fuse or cold fuse anything to it, do so now.
1. Invizuggle the item.
1. Uncull Mineru to uncull the zuggle.
1. Mount Mineru and fuse something over the zuggled item.

If you want to fuse entangle something to the item, you can Null FE to it.
