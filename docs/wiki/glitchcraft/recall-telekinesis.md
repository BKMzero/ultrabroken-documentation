# Recall Telekinesis

## Summary
Control recall locks or freecalls remotely by placing an object in any relative position you want them to go to.

_suusi - 30 September 2025_

## Instructions
### Basic Setup:
1. Freecall or recall lock any object.
2. Position a reference object (e.g., a hoverstone).
3. Recall the reference object (e.g., a hoverstone).
4. Glue freecall to another object before recall timer ends.

### Stable Setup using unbreakable glue (mulberry):
Same as above but permacull the freecall/recall lock after gluing it by either invizuggling its parent or sending its parent to culled purgatory.

## Notes
Warning: do not recall the glued-to object as that will break the glitch.
