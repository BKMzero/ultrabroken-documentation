# Graph View

An interactive map of how glitches reference each other across the Ultrabroken Archives.
Each node is a glitch entry; edges appear when one page mentions another.

- **Scroll** to zoom in/out
- **Click + drag** a node to reposition it
- **Ctrl + drag** (or middle-click drag) to pan
- **Click** a node to open its page
- **Hover** a node to highlight its connections
- On mobile: **pinch** to zoom, **drag** to pan
